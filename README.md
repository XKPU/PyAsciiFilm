# PyAsciiFilm

一个基于 Python 开发的创意工具，能够将视频实时转换为 ASCII 字符画，并直接在终端中进行流畅播放。同时支持将字符画视频导出为标准的视频文件。

## 运行时依赖库

运行本项目需要以下 Python 依赖：

- `numpy`
- `opencv-python`
- `pillow`
- `imageio-ffmpeg`
- `textual`
- `sounddevice`

## 构建依赖

如需将项目编译为可执行文件，需安装以下构建工具：

- `Nuitka`
- `zstandard`
