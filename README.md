# PyAsciiFilm

一个用 Python 编写的程序，可将视频转换为 ASCII 字符画，并直接在终端中播放。

**Python 依赖库：**
*   numpy
*   pillow
*   opencv-python

**外部依赖：**
*   需要安装 [FFmpeg](https://ffmpeg.org/)，并确保 `ffplay.exe` 已添加到系统的环境变量 `PATH` 中。
*   或者，您可以下载程序的 **"-with-ffplay"** 版本，该版本已包含 `ffplay`，无需额外配置环境。

---

A Python program that converts videos into ASCII art and plays them directly in the terminal.

**Python Dependencies:**
*   numpy
*   pillow
*   opencv-python

**External Dependency:**
*   Requires [FFmpeg](https://ffmpeg.org/) installed. Make sure `ffplay.exe` is available in your system's `PATH`.
*   Alternatively, download the **"-with-ffplay"** version of this program, which includes `ffplay` — no additional setup needed.
